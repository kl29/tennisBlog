Favorite Poly strings


Volkl Cyclone 1.30 / 1.25 (Yellow, Orange, Black)

10-sided "bite/spin" string. It's very crisp and lasts quite a while without notching. I string 55M / 50C.
I prefer the 1.25 version for more spin and bite, but it lasts way less long than 1.30. With this string you need to generate your own power, as it's rather dead. (It's very lively when brand new though!)
